# FAQ — Häufig gestellte Fragen

Hier finden Sie Antworten auf die häufigsten Fragen zu Vysible.

---

## Allgemein

### Was ist Vysible?

Vysible ist eine KI-gestützte Content-Automationsplattform für Arzt- und Zahnarztpraxen. Sie erstellt automatisch:
- Blogartikel für Ihre Website
- Newsletter für Ihre Patienten
- Social-Media-Posts für Facebook, Instagram & Co.

Alles basierend auf Ihrer Praxis-Website und Ihren Schwerpunkten.

### Für wen ist Vysible gedacht?

Vysible richtet sich an:
- **Marketing-Agenturen**, die Content für Arzt- und Zahnarztpraxen erstellen
- **Praxen**, die ihren Content selbst verwalten möchten

### Welche Fachgebiete werden unterstützt?

Aktuell unterstützte Fachgebiete:
- Zahnmedizin (Zahnarzt, Kieferorthopäde)
- Gynäkologie
- Allgemeinmedizin
- Weitere Fachgebiete auf Anfrage

## Projekte & Content-Generierung

### Wie lange dauert die Content-Generierung?

**15–25 Minuten** für ein 6-Monats-Paket (6 Themen mit je 1 Blogartikel, 1 Newsletter, 3 Social-Media-Posts).

Sie können das Fenster schließen — Sie erhalten eine E-Mail, sobald alles fertig ist.

### Kann ich die Generierung abbrechen?

Ja, während der Generierung sehen Sie einen **"Abbrechen"**-Button. Nach dem Abbruch werden keine Kosten berechnet.

### Was kostet die Content-Generierung?

Die Kosten hängen von Ihrem Agentur-Paket ab. Sie sehen die voraussichtlichen Kosten **vor** dem Start der Generierung im Dashboard.

**Transparenz:** Alle Kosten werden im **Kosten-Tracking** (Dashboard → Kosten) aufgeschlüsselt.

### Kann ich mehr als 6 Themen generieren?

Ja, Sie können beliebig viele Themen generieren. Klicken Sie nach der ersten Generierung auf **"Weitere Themen hinzufügen"**.

### Kann ich die Texte nachträglich ändern?

Ja, im **Editor** (Projekt → Inhalte → Thema auswählen → "Bearbeiten"). Sie können:
- Texte umschreiben
- Absätze hinzufügen/löschen
- Formatierungen anpassen

**Wichtig:** Änderungen werden **nicht** automatisch in bereits exportierten Dateien übernommen — exportieren Sie erneut nach Änderungen.

## Praxis-Portal & Freigabe

### Wie funktioniert die Praxis-Freigabe?

1. Sie laden die Praxis per E-Mail ein (Projekt → "Praxis einladen")
2. Die Praxis erhält einen **Freigabe-Link** (gültig 24 Stunden)
3. Die Praxis öffnet den Link und sieht alle Inhalte
4. Die Praxis kann jeden Inhalt **freigeben** oder **Kommentare** hinterlassen
5. Sie erhalten eine Benachrichtigung bei jeder Freigabe

**Mehr dazu:** [Praxis-Portal Anleitung](./praxis-portal.md)

### Wie lange ist der Freigabe-Link gültig?

**24 Stunden** ab Versand der E-Mail.

**Aber:** Nach dem ersten Öffnen bleibt die Praxis **7 Tage angemeldet** (Cookie) — sie kann den Link in dieser Zeit beliebig oft öffnen.

### Was passiert, wenn der Link abgelaufen ist?

Die Praxis sieht eine Fehlermeldung: *"Token abgelaufen"*.

**Lösung:** Senden Sie eine **neue Einladung** (Projekt → "Praxis einladen" → gleiche E-Mail-Adresse). Der neue Link ist wieder 24 Stunden gültig.

### Kann ich mehrere Praxis-Mitarbeiter einladen?

Ja, Sie können beliebig viele Personen einladen. Jede Person erhält einen **eigenen Freigabe-Link**.

**Wichtig:** Jeder Link ist personalisiert — Praxis-Mitarbeiter sollten Links **nicht untereinander teilen**.

### Sehe ich, wer was freigegeben hat?

Ja, im **Audit-Log** (Projekt → Einstellungen → Audit-Log) sehen Sie:
- Wer wann welchen Inhalt freigegeben hat
- Wer wann Kommentare geschrieben hat

## Export & Veröffentlichung

### Welche Export-Formate gibt es?

| Format | Verwendung |
|--------|------------|
| **DOCX** | Word-Dokument (zum Weiterbearbeiten) |
| **PDF** | Zum Ausdrucken oder Versenden |
| **XLSX** | Excel-Tabelle mit allen Texten |
| **HTML** | Für direktes Einfügen in Ihre Website |
| **ZIP** | Alle Formate zusammen |

### Kann ich nur freigegebene Inhalte exportieren?

Ja, beim Export können Sie wählen:
- **Nur freigegebene Inhalte** (empfohlen)
- **Alle Inhalte** (auch nicht freigegebene)

**Standard:** Nur freigegebene Inhalte.

### Kann Vysible direkt auf meiner Website veröffentlichen?

Ja, über **Integrationen**:
- **WordPress** — automatisches Veröffentlichen als Entwurf oder direkt live
- **Newsletter-Tools** — KlickTipp, Mailchimp (geplant)
- **Social Media** — Canva-Integration für Grafiken

**Mehr dazu:** [Benutzerhandbuch → Integrationen](./benutzerhandbuch.md#integrationen)

## Kosten & Abrechnung

### Wie werden die Kosten berechnet?

Vysible nutzt **KI-Modelle** (Anthropic Claude, OpenAI GPT), die nach **Token-Verbrauch** abgerechnet werden.

**Transparenz:** Sie sehen die Kosten **vor** jeder Generierung und können im **Kosten-Tracking** (Dashboard → Kosten) alle Ausgaben nachvollziehen.

### Kann ich ein Kosten-Limit setzen?

Ja, unter **Einstellungen → Kosten & Abrechnung** können Sie:
- **Monatliches Budget** festlegen
- **Warnung** bei 80 % des Budgets erhalten
- **Automatische Sperre** bei 100 % aktivieren

### Was passiert, wenn mein Budget aufgebraucht ist?

Sie können **keine neuen Generierungen** mehr starten, bis:
- Sie Ihr Budget erhöhen (Einstellungen → Kosten & Abrechnung)
- Der nächste Monat beginnt (Budget wird zurückgesetzt)

**Bestehende Projekte** bleiben lesbar — nur neue Generierungen sind gesperrt.

## Technische Fragen

### Welche Browser werden unterstützt?

| Browser | Unterstützung |
|---------|---------------|
| Chrome | ✅ Empfohlen |
| Firefox | ✅ Vollständig |
| Safari | ✅ Vollständig |
| Edge | ✅ Vollständig |
| Internet Explorer | ❌ Nicht unterstützt |

**Empfehlung:** Nutzen Sie die **neueste Version** Ihres Browsers.

### Funktioniert Vysible auf dem Smartphone?

**Eingeschränkt.** Vysible ist für **Desktop-Nutzung** optimiert.

**Auf dem Smartphone möglich:**
- Praxis-Portal (Freigabe-Link öffnen)
- Dashboard ansehen
- Projekte durchsuchen

**Nicht empfohlen:**
- Content-Generierung
- Editor-Nutzung
- Export-Funktionen

### Werden meine Daten verschlüsselt?

Ja, Vysible nutzt:
- **HTTPS** für alle Verbindungen (verschlüsselte Übertragung)
- **AES-256-GCM** für API-Keys und sensible Daten in der Datenbank
- **bcrypt** für Passwörter (12 Runden)

**Datenschutz:** Siehe [Datenschutzerklärung](https://vysible.cloud/datenschutz)

### Wo werden meine Daten gespeichert?

Alle Daten werden in der **EU** gespeichert (Deutschland, Hostinger VPS).

**Externe Dienste:**
- **KI-Modelle:** Anthropic (USA), OpenAI (USA) — nur Text-Verarbeitung, keine Speicherung
- **E-Mail:** SMTP-Server Ihrer Wahl (konfigurierbar)

## Fehlerbehebung

### Ich kann mich nicht einloggen

**Mögliche Ursachen:**

| Problem | Lösung |
|---------|--------|
| Falsches Passwort | Klicken Sie auf "Passwort vergessen" |
| E-Mail-Adresse falsch | Prüfen Sie Groß-/Kleinschreibung |
| Account gesperrt | Kontaktieren Sie den Support |

### Die Content-Generierung hängt

**Symptome:** Der Fortschrittsbalken bewegt sich nicht mehr.

**Lösung:**
1. Warten Sie **5 Minuten** — manchmal dauern einzelne Schritte länger
2. Aktualisieren Sie die Seite (F5) — der Fortschritt wird automatisch wiederhergestellt
3. Falls nach 30 Minuten keine Änderung: Kontaktieren Sie den Support

### Ich sehe keine Praxis-Kommentare im Dashboard

**Mögliche Ursachen:**

| Problem | Lösung |
|---------|--------|
| Keine neuen Kommentare | Der Badge erscheint nur bei **neuen** Kommentaren |
| Cookies deaktiviert | Aktivieren Sie Cookies in Ihrem Browser |
| Cache-Problem | Strg+F5 (Windows) / Cmd+Shift+R (Mac) |

### Der Export schlägt fehl

**Symptome:** Fehlermeldung beim Download.

**Lösung:**
1. Prüfen Sie, ob **mindestens ein Inhalt freigegeben** ist (bei "Nur freigegebene Inhalte")
2. Versuchen Sie ein **anderes Format** (z.B. PDF statt DOCX)
3. Leeren Sie Ihren **Browser-Cache** (Strg+Shift+Entf)
4. Falls weiterhin Fehler: Kontaktieren Sie den Support
