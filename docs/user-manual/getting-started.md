# Getting Started — Vysible

Willkommen bei Vysible! Diese Anleitung führt Sie Schritt für Schritt durch Ihre ersten Schritte.

---

## Was ist Vysible?

Vysible erstellt automatisch professionelle Content-Pakete für Arzt- und Zahnarztpraxen:
- **Blogartikel** für Ihre Website
- **Newsletter** für Ihre Patienten
- **Social-Media-Posts** für Facebook, Instagram & Co.

Alles basierend auf Ihrer Praxis-Website und Ihren Schwerpunkten — **6 Monate Content in wenigen Minuten**.

---

## Erste Schritte

### 1. Login

Öffnen Sie [https://vysible.cloud](https://vysible.cloud) in Ihrem Browser.

Geben Sie Ihre Zugangsdaten ein:
- **E-Mail:** Die E-Mail-Adresse, die Sie von uns erhalten haben
- **Passwort:** Ihr persönliches Passwort

**Tipp:** Beim ersten Login werden Sie aufgefordert, Ihr Passwort zu ändern.

---

### 2. Dashboard-Übersicht

Nach dem Login sehen Sie das **Dashboard** mit folgenden Bereichen:

| Bereich | Was Sie hier finden |
|---------|---------------------|
| **Projekte** | Liste aller Ihrer Praxen/Projekte |
| **Statistiken** | Anzahl generierter Inhalte, Kosten-Übersicht |
| **Praxis-Kommentare** | Neue Rückmeldungen von Praxis-Mitarbeitern (roter Punkt = neue Kommentare) |

---

### 3. Projekt anlegen

Ein **Projekt** entspricht einer Praxis. So legen Sie Ihr erstes Projekt an:

#### Schritt 1: Projekt erstellen

1. Klicken Sie auf **"Neues Projekt"** (oben rechts)
2. Geben Sie folgende Informationen ein:

   | Feld | Beispiel | Erklärung |
   |------|----------|-----------|
   | **Name** | Dr. Müller Zahnmedizin | Interner Name (wird nicht veröffentlicht) |
   | **Praxis-URL** | https://dr-mueller-zahnarzt.de | Ihre Website-Adresse |
   | **Fachgebiet** | Zahnarzt | Wählen Sie aus: Zahnarzt, Kieferorthopäde, Gynäkologe, ... |

3. Klicken Sie auf **"Projekt anlegen"**

#### Schritt 2: Positionierung definieren

Vysible öffnet automatisch das **Positionierungs-Interview** (Hedy):

1. Beantworten Sie 5–7 Fragen zu Ihrer Praxis:
   - Welche Behandlungen bieten Sie an?
   - Wer ist Ihre Zielgruppe?
   - Was unterscheidet Sie von anderen Praxen?

2. **Beispiel-Antworten:**
   ```
   Frage: Welche Behandlungen bieten Sie an?
   Antwort: Prophylaxe, Implantologie, ästhetische Zahnmedizin, 
            Kinderzahnheilkunde
   
   Frage: Wer ist Ihre Zielgruppe?
   Antwort: Familien mit Kindern, Angstpatienten, Senioren
   ```

3. Klicken Sie auf **"Positionierung speichern"**

#### Schritt 3: Themen festlegen

1. Geben Sie **6 Themen** ein, über die Sie schreiben möchten:

   **Beispiel (Zahnarzt):**
   - Professionelle Zahnreinigung — warum sie wichtig ist
   - Zahnimplantate: Ablauf und Kosten
   - Angst vor dem Zahnarzt überwinden
   - Zahngesundheit bei Kindern
   - Bleaching: Methoden im Vergleich
   - Parodontitis erkennen und behandeln

2. **Tipp:** Vysible schlägt automatisch Themen vor — Sie können diese übernehmen oder anpassen.

3. Klicken Sie auf **"Themen speichern"**

---

### 4. Content generieren

Jetzt kommt der spannende Teil — die KI erstellt Ihre Inhalte:

1. Klicken Sie auf **"Content generieren"**
2. Vysible erstellt nun:
   - 6 Blogartikel (je 800–1200 Wörter)
   - 6 Newsletter (je 400–600 Wörter)
   - 18 Social-Media-Posts (3 pro Thema)

**Dauer:** 15–25 Minuten (Sie können das Fenster schließen — Sie erhalten eine E-Mail, sobald alles fertig ist)

---

### 5. Ergebnisse ansehen

Nach der Generierung:

1. Öffnen Sie Ihr Projekt
2. Klicken Sie auf **"Inhalte"**
3. Sie sehen alle generierten Texte in einer Übersicht:

   | Spalte | Bedeutung |
   |--------|-----------|
   | **Thema** | Das gewählte Thema |
   | **Blog** | Blogartikel (Vorschau) |
   | **Newsletter** | Newsletter-Text (Vorschau) |
   | **Social** | Social-Media-Posts (Anzahl) |
   | **Status** | Ausstehend / Freigegeben |

4. Klicken Sie auf ein Thema, um die vollständigen Texte zu lesen

---

### 6. Praxis-Freigabe einholen

Bevor Sie die Inhalte veröffentlichen, holen Sie die Freigabe der Praxis ein:

1. Klicken Sie auf **"Praxis einladen"** (oben rechts im Projekt)
2. Geben Sie ein:
   - **Name:** Dr. Anna Müller
   - **E-Mail:** anna.mueller@praxis-beispiel.de

3. Klicken Sie auf **"Einladung senden"**

Die Praxis erhält eine E-Mail mit einem **Freigabe-Link** (gültig 24 Stunden).

**Was passiert dann?**
- Die Praxis öffnet den Link
- Sie sieht alle Inhalte in einer übersichtlichen Ansicht
- Sie kann jeden Inhalt **freigeben** oder **Kommentare** hinterlassen
- Sie erhalten eine Benachrichtigung bei jeder Freigabe

**Mehr dazu:** Siehe [Praxis-Portal Anleitung](./praxis-portal.md)

---

### 7. Content exportieren

Sobald die Inhalte freigegeben sind:

1. Öffnen Sie Ihr Projekt → **"Inhalte"**
2. Klicken Sie auf **"Exportieren"**
3. Wählen Sie das gewünschte Format:
   - **DOCX** — für Word (zum Weiterbearbeiten)
   - **PDF** — zum Ausdrucken oder Versenden
   - **XLSX** — Excel-Tabelle mit allen Texten
   - **HTML** — für direktes Einfügen in Ihre Website
   - **ZIP** — alle Formate zusammen

4. Klicken Sie auf **"Download starten"**

## Häufige Fragen (Schnellantworten)

**Wie lange dauert die Content-Generierung?**  
→ 15–25 Minuten für ein 6-Monats-Paket (6 Themen)

**Kann ich die Texte nachträglich ändern?**  
→ Ja, im Editor (siehe [Benutzerhandbuch](./benutzerhandbuch.md#content-bearbeiten))

**Was kostet die Generierung?**  
→ Siehe Ihr Agentur-Paket — die Kosten werden im Dashboard angezeigt

**Wie oft kann ich Praxis-Einladungen versenden?**  
→ Unbegrenzt — jede Einladung ist 24 Stunden gültig
