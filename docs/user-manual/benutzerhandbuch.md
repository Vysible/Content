## Dashboard

Das **Dashboard** ist Ihre Startseite nach dem Login.

### Übersicht

| Bereich | Inhalt |
|---------|--------|
| **Projekte** | Liste aller Ihrer Praxen/Projekte |
| **Statistiken** | Anzahl generierter Inhalte, Gesamtkosten |
| **Praxis-Kommentare** | Neue Rückmeldungen (roter Badge = neue Kommentare) |
| **Letzte Aktivitäten** | Ihre letzten Aktionen (Generierungen, Freigaben) |

### Schnellaktionen

- **Neues Projekt** — Oben rechts, blaue Schaltfläche
- **Projekt öffnen** — Klick auf Projekt-Karte
- **Kosten ansehen** — Sidebar → "Kosten & Abrechnung"

## Projekte verwalten

### Projekt anlegen

**Schritt 1:** Klicken Sie auf **"Neues Projekt"**

**Schritt 2:** Füllen Sie das Formular aus:

| Feld | Pflicht | Beispiel |
|------|---------|----------|
| **Name** | Ja | Dr. Müller Zahnmedizin |
| **Praxis-URL** | Ja | https://dr-mueller-zahnarzt.de |
| **Fachgebiet** | Ja | Zahnarzt |
| **Beschreibung** | Nein | Familienpraxis in München |

**Schritt 3:** Klicken Sie auf **"Projekt anlegen"**

**Was passiert dann?**
- Vysible öffnet automatisch das **Positionierungs-Interview** (Hedy)
- Sie beantworten 5–7 Fragen zu Ihrer Praxis
- Vysible speichert Ihre Antworten und schlägt Themen vor

### Projekt bearbeiten

1. Öffnen Sie das Projekt
2. Klicken Sie auf **"Einstellungen"** (Zahnrad-Symbol oben rechts)
3. Ändern Sie die gewünschten Felder
4. Klicken Sie auf **"Speichern"**

**Änderbare Felder:**
- Name
- Beschreibung
- Positionierung (erneutes Interview)

**Nicht änderbar:**
- Praxis-URL (aus technischen Gründen)
- Fachgebiet (aus technischen Gründen)

### Projekt löschen

**Achtung:** Das Löschen eines Projekts ist **endgültig** und kann nicht rückgängig gemacht werden!

1. Öffnen Sie das Projekt
2. Klicken Sie auf **"Einstellungen"**
3. Scrollen Sie nach unten zu **"Gefahrenzone"**
4. Klicken Sie auf **"Projekt löschen"**
5. Bestätigen Sie mit Eingabe des Projekt-Namens

**Was wird gelöscht?**
- Alle generierten Inhalte
- Alle Freigaben und Kommentare
- Alle Export-Dateien
- Alle Einladungen

## Content-Generierung

### Vorbereitung

Bevor Sie Content generieren, benötigen Sie:
1. **Positionierung** — Abgeschlossenes Hedy-Interview
2. **Themen** — Mindestens 1 Thema (empfohlen: 6 für ein 6-Monats-Paket)

### Themen festlegen

**Schritt 1:** Öffnen Sie Ihr Projekt → **"Themen"**

**Schritt 2:** Klicken Sie auf **"Thema hinzufügen"**

**Schritt 3:** Geben Sie ein Thema ein:

**Beispiele (Zahnarzt):**
- Professionelle Zahnreinigung — warum sie wichtig ist
- Zahnimplantate: Ablauf und Kosten
- Angst vor dem Zahnarzt überwinden
- Zahngesundheit bei Kindern
- Bleaching: Methoden im Vergleich
- Parodontitis erkennen und behandeln

**Tipp:** Vysible schlägt automatisch Themen vor basierend auf:
- Ihrer Praxis-Website
- Ihrer Positionierung
- Ihrem Fachgebiet

**Schritt 4:** Wiederholen Sie für alle gewünschten Themen (empfohlen: 6)

**Schritt 5:** Klicken Sie auf **"Themen speichern"**

### Generierung starten

**Schritt 1:** Klicken Sie auf **"Content generieren"**

**Schritt 2:** Prüfen Sie die Vorschau:

| Was wird generiert | Anzahl |
|-------------------|--------|
| Blogartikel | 6 (je 800–1200 Wörter) |
| Newsletter | 6 (je 400–600 Wörter) |
| Social-Media-Posts | 18 (3 pro Thema) |

**Schritt 3:** Prüfen Sie die **voraussichtlichen Kosten**

**Schritt 4:** Klicken Sie auf **"Jetzt generieren"**

### Während der Generierung

Sie sehen einen **Fortschrittsbalken** mit folgenden Schritten:

1. **Scraping** — Vysible liest Ihre Praxis-Website
2. **Themen-Analyse** — KI analysiert Ihre Themen
3. **Blog-Generierung** — Blogartikel werden erstellt
4. **Newsletter-Generierung** — Newsletter werden erstellt
5. **Social-Media-Generierung** — Posts werden erstellt
6. **Finalisierung** — Alles wird gespeichert

**Dauer:** 15–25 Minuten

**Tipp:** Sie können das Fenster schließen — Sie erhalten eine E-Mail, sobald alles fertig ist.

### Generierung abbrechen

Während der Generierung sehen Sie einen **"Abbrechen"**-Button.

**Wichtig:**
- Nach dem Abbruch werden **keine Kosten** berechnet
- Bereits generierte Inhalte werden **nicht** gespeichert
- Sie können jederzeit eine neue Generierung starten

## Content bearbeiten

### Editor öffnen

1. Öffnen Sie Ihr Projekt → **"Inhalte"**
2. Klicken Sie auf ein Thema
3. Klicken Sie auf **"Bearbeiten"** (Stift-Symbol)

### Editor-Funktionen

Der Editor bietet folgende Funktionen:

| Funktion | Tastenkombination | Beschreibung |
|----------|-------------------|--------------|
| **Fett** | Strg+B | Text fett formatieren |
| **Kursiv** | Strg+I | Text kursiv formatieren |
| **Überschrift** | Strg+Alt+1–3 | Überschrift Ebene 1–3 |
| **Liste** | Strg+Shift+8 | Aufzählungsliste |
| **Nummerierung** | Strg+Shift+7 | Nummerierte Liste |
| **Rückgängig** | Strg+Z | Letzte Änderung rückgängig |
| **Wiederholen** | Strg+Y | Rückgängig gemachte Änderung wiederholen |

### Änderungen speichern

**Automatisches Speichern:** Alle 30 Sekunden

**Manuelles Speichern:** Klicken Sie auf **"Speichern"** (oben rechts)

**Indikator:** Sie sehen einen **grünen Haken** bei erfolgreicher Speicherung

### Versionen

Jede Speicherung erstellt eine **neue Version**.

**Versionen ansehen:**
1. Klicken Sie auf **"Versionen"** (Uhr-Symbol)
2. Sie sehen alle gespeicherten Versionen mit Zeitstempel
3. Klicken Sie auf eine Version, um sie anzusehen
4. Klicken Sie auf **"Wiederherstellen"**, um zu dieser Version zurückzukehren

**Wichtig:** Wiederherstellung überschreibt die aktuelle Version — erstellen Sie vorher eine Sicherung!

## Content-Kalender

Der **Content-Kalender** hilft Ihnen, Ihre Veröffentlichungen zu planen.

### Kalender öffnen

1. Öffnen Sie Ihr Projekt
2. Klicken Sie auf **"Kalender"** (Kalender-Symbol in der Sidebar)

### Inhalt planen

**Schritt 1:** Klicken Sie auf ein Datum im Kalender

**Schritt 2:** Wählen Sie einen Inhalt aus:
- Blogartikel
- Newsletter
- Social-Media-Post

**Schritt 3:** Klicken Sie auf **"Planen"**

**Ergebnis:** Der Inhalt erscheint im Kalender am gewählten Datum

### Planung ändern

**Verschieben:**
1. Klicken Sie auf den geplanten Inhalt
2. Ziehen Sie ihn auf ein neues Datum (Drag & Drop)

**Löschen:**
1. Klicken Sie auf den geplanten Inhalt
2. Klicken Sie auf **"Planung entfernen"**

### Kalender-Ansichten

| Ansicht | Beschreibung |
|---------|--------------|
| **Monat** | Übersicht über einen ganzen Monat |
| **Woche** | Detailansicht einer Woche |
| **Tag** | Alle Inhalte eines Tages |

**Wechseln:** Klicken Sie auf die entsprechende Schaltfläche oben rechts

## Export-Funktionen

### Export starten

1. Öffnen Sie Ihr Projekt → **"Inhalte"**
2. Klicken Sie auf **"Exportieren"** (Download-Symbol oben rechts)

### Format wählen

| Format | Verwendung | Inhalt |
|--------|------------|--------|
| **DOCX** | Word-Dokument | Alle Texte formatiert |
| **PDF** | Ausdrucken/Versenden | Alle Texte als PDF |
| **XLSX** | Excel-Tabelle | Alle Texte in Tabellenform |
| **HTML** | Website-Einfügung | Alle Texte als HTML |
| **ZIP** | Alle Formate | Alle Formate in einem Archiv |

### Filter

**Nur freigegebene Inhalte** (Standard):
- Nur Inhalte mit Status "Freigegeben" werden exportiert
- Empfohlen für die Veröffentlichung

**Alle Inhalte**:
- Auch nicht freigegebene Inhalte werden exportiert
- Nützlich für interne Prüfungen

### Download

Nach der Auswahl:
1. Klicken Sie auf **"Download starten"**
2. Der Download beginnt automatisch
3. Die Datei wird in Ihrem Downloads-Ordner gespeichert

**Dateiname:** `vysible-export-[projektname]-[datum].zip`

## Praxis-Freigabe

### Praxis einladen

**Schritt 1:** Öffnen Sie Ihr Projekt

**Schritt 2:** Klicken Sie auf **"Praxis einladen"** (oben rechts)

**Schritt 3:** Füllen Sie das Formular aus:

| Feld | Beispiel |
|------|----------|
| **Name** | Dr. Anna Müller |
| **E-Mail** | anna.mueller@praxis-beispiel.de |

**Schritt 4:** Klicken Sie auf **"Einladung senden"**

**Was passiert?**
- Die Praxis erhält eine E-Mail mit einem **Freigabe-Link**
- Der Link ist **24 Stunden** gültig
- Nach dem ersten Öffnen bleibt die Praxis **7 Tage** angemeldet

### Freigaben verfolgen

**Dashboard-Badge:**
- Roter Punkt bei **"Praxis-Kommentare"** = neue Kommentare
- Zahl zeigt Anzahl der neuen Kommentare

**Projekt-Übersicht:**
1. Öffnen Sie Ihr Projekt → **"Inhalte"**
2. Sie sehen den Status jedes Inhalts:
   - **Ausstehend** (grau) — noch nicht freigegeben
   - **Freigegeben** (grün) — von der Praxis freigegeben

**Kommentare ansehen:**
1. Klicken Sie auf einen Inhalt
2. Scrollen Sie nach unten zu **"Kommentare"**
3. Sie sehen alle Kommentare der Praxis und Ihre Antworten

### Auf Kommentare antworten

1. Öffnen Sie den Inhalt mit Kommentaren
2. Geben Sie Ihre Antwort in das Textfeld ein
3. Klicken Sie auf **"Antworten"**

**Wichtig:** Die Praxis erhält **keine automatische Benachrichtigung** — sie muss den Freigabe-Link erneut öffnen, um Ihre Antwort zu sehen.

## Integrationen

Vysible kann sich mit externen Diensten verbinden.

### WordPress

**Funktion:** Automatisches Veröffentlichen von Blogartikeln auf Ihrer WordPress-Website.

**Einrichtung:**

1. Öffnen Sie **Einstellungen → Integrationen → WordPress**
2. Geben Sie ein:
   - **WordPress-URL:** https://ihre-website.de
   - **Benutzername:** Ihr WordPress-Admin-Benutzername
   - **Application-Password:** Generieren Sie ein Application-Password in WordPress (Benutzer → Profil → Application-Passwords)
3. Klicken Sie auf **"Verbindung testen"**
4. Bei Erfolg: Klicken Sie auf **"Speichern"**

**Nutzung:**
1. Öffnen Sie einen Blogartikel
2. Klicken Sie auf **"Auf WordPress veröffentlichen"**
3. Wählen Sie:
   - **Als Entwurf** — Artikel wird als Entwurf gespeichert
   - **Sofort veröffentlichen** — Artikel geht direkt live
4. Klicken Sie auf **"Veröffentlichen"**

### Canva

**Funktion:** Erstellen Sie Grafiken für Social-Media-Posts.

**Einrichtung:**

1. Öffnen Sie **Einstellungen → Integrationen → Canva**
2. Klicken Sie auf **"Mit Canva verbinden"**
3. Sie werden zu Canva weitergeleitet — melden Sie sich an
4. Klicken Sie auf **"Zugriff erlauben"**
5. Sie werden zurück zu Vysible geleitet — die Verbindung ist hergestellt

**Nutzung:**
1. Öffnen Sie einen Social-Media-Post
2. Klicken Sie auf **"Grafik erstellen"**
3. Canva öffnet sich mit einer Vorlage
4. Bearbeiten Sie die Grafik in Canva
5. Klicken Sie auf **"Zurück zu Vysible"**
6. Die Grafik wird automatisch mit dem Post verknüpft

### KlickTipp (Newsletter)

**Funktion:** Versenden Sie Newsletter direkt aus Vysible.

**Einrichtung:**

1. Öffnen Sie **Einstellungen → Integrationen → KlickTipp**
2. Geben Sie ein:
   - **Benutzername:** Ihr KlickTipp-Benutzername
   - **Passwort:** Ihr KlickTipp-Passwort
3. Klicken Sie auf **"Verbindung testen"**
4. Bei Erfolg: Klicken Sie auf **"Speichern"**

**Nutzung:**
1. Öffnen Sie einen Newsletter
2. Klicken Sie auf **"An KlickTipp senden"**
3. Wählen Sie die **Empfänger-Liste**
4. Klicken Sie auf **"Senden"**

## Einstellungen

### Profil

**Passwort ändern:**
1. Öffnen Sie **Einstellungen → Profil**
2. Geben Sie ein:
   - **Aktuelles Passwort**
   - **Neues Passwort**
   - **Neues Passwort wiederholen**
3. Klicken Sie auf **"Passwort ändern"**

**E-Mail-Adresse ändern:**
1. Öffnen Sie **Einstellungen → Profil**
2. Geben Sie Ihre neue E-Mail-Adresse ein
3. Klicken Sie auf **"E-Mail ändern"**
4. Sie erhalten eine Bestätigungs-E-Mail — klicken Sie auf den Link

### API-Keys

Hier verwalten Sie Ihre KI-API-Keys (Anthropic, OpenAI).

**Wichtig:** Diese Einstellung ist nur für **Agentur-Administratoren** sichtbar.

### Kosten & Abrechnung

**Budget festlegen:**
1. Öffnen Sie **Einstellungen → Kosten & Abrechnung**
2. Geben Sie Ihr **monatliches Budget** ein (z.B. 500 €)
3. Aktivieren Sie **"Warnung bei 80 %"**
4. Optional: Aktivieren Sie **"Automatische Sperre bei 100 %"**
5. Klicken Sie auf **"Speichern"**

**Kosten-Übersicht:**
- **Aktueller Monat:** Ausgaben im laufenden Monat
- **Letzte 30 Tage:** Rollierender 30-Tage-Durchschnitt
- **Pro Projekt:** Aufschlüsselung nach Projekten
